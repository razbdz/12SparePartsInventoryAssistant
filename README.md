Spare Parts Inventory Assistant er et C#-konsolprogram, der lader brugeren søge efter reservedele i et lille lager. Programmet bliver ved med at spørge, 
indtil brugeren indtaster en reservedel, der findes. 
Det kan også vise hele lageret, hvis brugeren spørger, om der overhovedet er nogle dele på lager. 
Programmet demonstrerer brug af arrays, loops og strengsammenligning i C#.
